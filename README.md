# CodeAlpha Data Analytics Internship Project

## Project Title

Netflix Data Analysis

## Description

In this project I performed basic data analysis on the Netflix dataset.
The goal of this project was to understand the dataset, explore patterns, and create simple visualizations.

This project includes:

- Exploratory Data Analysis (EDA)
- Data Visualization

## Dataset

Netflix Movies and TV Shows Dataset

## Tasks Performed

### 1. Exploratory Data Analysis

In this step I explored the dataset to understand its structure.

Operations performed:

- Loaded the dataset using pandas
- Displayed first few rows of the dataset
- Checked dataset shape and columns
- Checked missing values
- Generated statistical summary using "describe()"

### 2. Data Visualization

I created basic visualizations to understand the distribution of data.

Visualizations created:

- Bar chart for content type
- Histogram for release year
- Pie chart for content distribution
- Heatmap for missing values

### 3. Advanced EDA Tasks

I added further exploration to deep dive into specific aspects of the dataset:

- **Content Rating Analysis**: Created a countplot to analyze the distribution of age and maturity ratings across content.
- **Genre Distribution**: Analyzed the `listed_in` column to determine the Top 10 most common genres/categories.
- **Data Filtering & Extraction**: Demonstrated how to filter the dataset for very specific criteria (e.g., finding all Indian Movies).

Visualizations created:

- Bar chart for content ratings (Order of popularity)
- Bar chart for Top 10 genres

## Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn
